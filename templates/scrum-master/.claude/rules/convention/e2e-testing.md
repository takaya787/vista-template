# E2E Testing

This scrum-master template uses Playwright only for Notion browser automation (see `rules/convention/integrations.md`).

Codebase E2E test standards (Playwright spec authoring, selectors, directory layout) belong in the **engineer** template at `templates/engineer/.claude/rules/`.

If this project co-locates engineer and scrum-master rules, copy `templates/engineer/.claude/rules/convention/coding-standards.md` and the Playwright conventions there — do not maintain them here.
