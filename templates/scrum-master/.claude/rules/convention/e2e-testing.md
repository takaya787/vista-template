# E2E Testing

This scrum-master template uses Playwright only for Notion browser automation (see `rules/convention/integrations.md`).

Codebase E2E test standards (Playwright spec authoring, selectors, directory layout) are outside the scope of this template.

If this project needs E2E test conventions, add them as project-specific rules in `.claude/rules/config/` — do not maintain them here.
